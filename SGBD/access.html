Basicamente, o Microsoft Access é um sistema de gerenciamento de banco de dados da Microsoft.

Seu desenvolvedor foi a própria Microsoft, tendo como versão mais atual:

  • 2019 (16.0) / (24 de setembro de 2018)

Sua licença para utilização é chamada de 'Shareware', ou seja, por princípio é disponibilizada gratuitamente, podendo custear algumas funcionalidades ou até mesmo depois de determinado tempo de uso.
